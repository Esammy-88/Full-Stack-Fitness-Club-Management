About This Project

The Health & Fitness Club Management System is a comprehensive web application developed to demonstrate my full-stack development skills. This project showcases my proficiency in database design, backend development with Python/Flask, and modern frontend design with Bootstrap.

Database Architecture

The system utilizes a normalized relational database design with the following key entities:

  • Members - User accounts and profiles
  • Trainers - Staff and specializations
  • Classes - Group fitness sessions
  • Personal Training - 1-on-1 sessions
  • Fitness Goals - Member objectives
  • Health Metrics - Progress tracking
  • Rooms - Facility management
  • Equipment - Asset tracking
  • Billing - Payment processing
  • Payments - Transaction records
  • Availability - Trainer schedules
  • Registrations - Class enrollments
Includes advanced features like database triggers, views, indexes, and constraints for data integrity.
